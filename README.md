# Guia
* Para instalar dependências digite no terminal oque esta  aqui ⬇️

```
npm init
docker-compose up
```
* Abra um novo terminal para iniciarmos o servidor ⬇️
```npm
npm start
```
* Após isso basta ir em [http://localhost:3000](http://localhost:3000)
